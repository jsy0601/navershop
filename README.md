# navershop
### 주요 기능
1. 키워드로 최저가 상품 검색
  - 네이버 쇼핑 API 이용
2. 관심 상품 등록하기
3. 등록된 관심 상품의 희망 최저가 설정하기
4. 등록된 관심 상품 조회하기
5. 등록된 관심 상품 최저가 정보 매일 업데이트

### skills
- Spring Web / Spring Data JPA / H2 Database / MySQL Driver

- Type: Gradle
- Language: Java
- Java Version: 8
